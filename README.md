# 像素螺丝工匠 - Pixel Screwcrafter

一款通过拆卸与重组螺丝来还原复古像素艺术的休闲解压拼图游戏。

## 🎮 游戏简介

在这个游戏中，玩家扮演一位技艺高超的螺丝工匠，通过拧开并重新组合螺丝，来逐步完成一幅幅精美的像素艺术作品。

### 游戏特色

- **沉浸在怀旧的像素艺术之中** - 每一块拼图都是精心设计的艺术品
- **多样化的主题系列** - 从自然景观到经典角色，从科幻场景到日常物品，各种系列等待您的解锁与收藏
- **轻松上手** - 简单直观的操作方式让任何人都能快速上手
- **渐进式挑战** - 随着关卡推进，考验玩家的观察力和耐心
- **舒缓的游戏体验** - 轻柔的游戏音乐配合舒缓的视觉效果，为玩家提供了一个完美的休闲空间

## 🎯 游戏玩法

1. **拧下螺丝** - 点击画布上的螺丝将其拧下，螺丝会进入螺丝托盘
2. **选择螺丝** - 从螺丝托盘中选择一个螺丝
3. **放置螺丝** - 点击画布上的空位将螺丝拧上
4. **完成拼图** - 将所有螺丝放回正确位置即可还原像素艺术作品

### 游戏机制

- **计时器** - 记录完成关卡所用时间
- **移动计数** - 记录移动螺丝的次数
- **星级评分** - 根据用时和移动次数给出1-3星评价
- **撤销功能** - 可以撤销最近的操作
- **提示功能** - 如果遇到困难可以使用提示
- **关卡解锁** - 完成当前关卡后自动解锁下一关

## 🚀 快速开始

### 环境要求

- Node.js (用于运行服务器)

### 安装与运行

1. **进入项目目录**
```bash
cd /home/ubuntu/Pixel-Screwcrafter
```

2. **启动服务器**
```bash
node server.js
```

3. **访问游戏**

打开浏览器，访问：
```
http://localhost:2421
```

### 其他运行方式

如果不想使用Node.js，也可以直接在浏览器中打开index.html文件：

```bash
# 使用Python 3启动简单服务器
python3 -m http.server 2421

# 或使用Python 2
python -m SimpleHTTPServer 2421
```

## 💾 数据存储

游戏数据会自动保存到浏览器的localStorage中：

- 已解锁的关卡
- 游戏统计（完成关卡数、拆卸螺丝数、完美通关数）
- 设置选项（音量、音效开关、主题）

## 🎯 评分系统

完成关卡后会根据以下条件给出星级评分：

- **1星**：完成关卡即可获得
- **2星**：移动次数不超过完美移动数
- **3星**：同时满足用时不超过目标时间和移动次数不超过完美移动数

## 📝 开发说明

### 添加新关卡

在 `levels.js` 中添加新的关卡数据：

```javascript
{
    id: 'unique_id',
    name: '关卡名称',
    emoji: '🎨',
    difficulty: '简单',
    difficultyLevel: 1,
    gridSize: 12,
    screwCount: 8,
    unlocked: true,
    targetTime: 120,
    perfectMoves: 10,
    pixels: [
        // 像素数据数组
    ],
    colorMap: {
        // 颜色映射
    },
    screwPositions: [
        // 螺丝位置
    ]
}
```

### 修改服务器端口

在 `server.js` 中修改 PORT 常量：

```javascript
const PORT = 2421; // 修改为其他端口
```
